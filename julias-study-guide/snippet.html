  • "p" denotes a paragraph and contains plain text. The browser automatically displays a space before and after the paragraph. It is also important to remember that any extra spaces and lines in a paragraph will not be displayed.
  • "a" denotes a link element. Since it is contained within the "p" tag, we say that a is the child of p. This relationship is useful in structuring information (look up the document tree.)
  • Re: opening and closing tags - "p" is at the start of the content while "/p" is at the end. The first instance of "p" is an opening tag, and it appears at the start of the content. "/p" is a closing tag and appears at the end. If you don't include both tags, the computer won't be able to read your code and the text won't show up, so make sure to include them! All tags except "br" and "hr" have an open tag and a close tag. "a" has one as well, and it contains the text that the user clicks on to be redirected to the linked location.
  • Attributes: href, id, and class (contained within the <> of the opening tag for "a") are called attributes. href is used to connect the link. id and class are labels that we will need when we are writing css and want to target a specific section of text. They are specifiers. For now, it is important to know that id is more specific than class. There is another attribute, target, and target="_blank" will tell the link to open in a new tab.
