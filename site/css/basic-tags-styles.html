Block-level text containers

Block-level elements appear on the screen as if they have a line break before and after them. They all start on their own new line. (Source: www.tutorialspoint.com/html)

  • "p"

    • The p element identifies blocks of text. Although the closing tag technically isn't necessary, it's good practice to use it. An important detail is that content in the p tag will not indent or include line breaks! It will come out as strings of sentences. Formatting paragraphs will be addressed with CSS, which we will talk about later.

  • "blockquote"

    • The blockquote element is used for quotations (i.e. quotations that are longer than one line. Browsers generally show block quotes as indented text.

  • "pre"

    • The pre element displays as much space as specified. It tells the browser to preserve both spaces and line breaks.

Inline text styling

Inline elements can appear within sentences and do not have to appear on a new line of their own. (Source: www.tutorialspoint.com/html)

  • "em" and "i"

    • i is used for text in an "alternate voice" (e.g. foreign words, technical terms, etc.) and em is used to put stress/emphasis on a word.

  • "strong" and "b"

    • b is used to stylistically offset words, e.g. keywords/definitions, roles in a play, etc. strong is used to represent contents' strong importance.

  • "code"

    • The code element displays content in a style that indicated the text is a short fragment of computer code. Basically, it turns "code" into code if you want to indicate that it is computer code.

The "a" tag

The a element creates a hyperlink to web pages, files, email adresses, locations in the same page, or anything else a url can address. The attribute href is used to specify the link's destination.

The "img" tag

The img element is used to embed an image in an HTML page. The attribute src is used to specify the path to the image (i.e. how to browser knowns what image to display) and the attribute alt specifies an alternative text for the image if, for some reason, it can't be displayed. The alt attribute is necessary for you website to be verified, so include it! You can also specify the height and width of the image using the attributes height and width.

Heading tags (h1-h6)

h1 specifies the most important heading and is used at the top of your page. It is important to use headings in your document to give it structure. Notice how the headings in this page help you organize information faster.

Ordered and unordered lists

The ul element denotes and unorder list. The browser automatically indents this text. Within the parent element ul, use the element li to denote bulleted items.

The ol element is similar, except it numbers list items instead of bulleting them.
