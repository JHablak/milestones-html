What is CSS?

CSS is short for cascading style sheets. Whereas HTML is used to write content for your page, CSS is used to style it. You can use it to change the background of your page, change the font and font-size, add borders, and much much more. There are three ways to add CSS to your html page:

  1. Inline - CSS applied individually to each element
  2. Internal CSS - CSS consolidated in head. This method is good for writing CSS for one page.
  3. External CSS - CSS written in a separate file and linked to the head of each HTML page.

To style this page, I am going to use internal CSS.

Some examples of how properties work:

  1. color is used to change the color of text on the page. I used the color property to make this text red.
  2. text-align can be specified to "left," "right," "center," or "justify." This line is centered!
  3. text-transformation controls the capitalization of text and can be specified as "uppercase," "lowercase," or "capitalize" (also none, inherit, or initial.)
  4. line-height changes the height of the line. It includes the text itself and the space below it or above & below it. (This line has a line height of 55px! But you should see what happens when you change it to 0.)
  5. letter-spacing ... woah, guess what this does :)
  6. font-family changes the font. Make sure to use backup font unless the one you want isn't available on a certain page.
  7. font-size changes the size of the text.
  8. font-style can be normal, italic or oblique.
  9. font-weight specifies the boldness of the text.
  10. background-color changes the background color of the element!
  11. background-image make the background of an element an image...this is getting crazier & crazier.
  12. list-style-type changes the bullet points. There's not one on this list item.
  13. width and height For an image, this specifies the width and height.
  14. margin changes the space around the element. (I'm using background color to better illustrate this.)
  15. padding changes the space around the content. (Ditto the parenthetical above.)
  16. border creates a border around the element.
